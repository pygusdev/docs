Política de Privacidade da PYGUS

Esta política de privacidade detalha o tratamento dos dados dos usuários (doravante denominados 'Pacientes') que utilizam a Plataforma PYGUS, oferecida pela LAMAR Ltda. Para a PYGUS, é de suma importância preservar a privacidade e a segurança dos Dados Pessoais dos Pacientes.

1. RESPONSÁVEL DOS DADOS

O Responsável pelo tratamento dos dados dos Pacientes coletados através deste aplicativo é a PYGUS, cujos dados são os seguintes:

  • Nome: LAMAR Ltda (doravante, "PYGUS")
  • Encarregado de Proteção de Dados (DPO): PYGUS
  • Dados de registro: CNPJ/MF sob o nº 47.932.823/0001-19
  • Domicílio: Rua Florestal, 57, Chã da Jaqueira, CEP 57018-528, Maceió-Alagoas
  • E-mail para contato: suporte@pygus.com.br

2. DADOS QUE SÃO COLETADOS

PYGUS coleta dados pessoais dos pacientes a fim de fornecer os melhores serviços possíveis e possibilitar o desenvolvimento da fala. Os tipos de dados coletados incluem, mas não se limitam a:

  • Dados cadastrais: nome completo, data de nascimento, telefone do responsável, endereço de e-mail
  • Dados de uso da plataforma: informações de todas as interações realizadas dentro do aplicativo

3. FINALIDADES E LEGITIMAÇÃO

A PYGUS trata os dados do Paciente com um objetivo específico e são tratados apenas os dados que são pertinentes para cumprir esse objetivo. Neste sentido, os dados poderão ser tratados para os seguintes propósitos:

  • Prestação adequada dos serviços:
    • Criar uma conta
    • Melhoria contínua dos serviços, através de análises e estudos comportamentais dos usuários
  • Utilizar seus dados para gerar e compartilhar informação agregada que não o identifique para fins analíticos e estatísticos
  • Prevenção de abuso e fraude no uso de nossos serviços
  • Cessão de dados a organismos e autoridades públicas, sempre em conformidade com as disposições legais e regulamentares

4. PRAZO DE CONSERVAÇÃO DOS DADOS

Os dados dos Pacientes são mantidos enquanto a conta permanecer ativa ou enquanto necessário para a prestação dos serviços. Após esse período, os dados serão excluídos ou anonimizados de acordo com as políticas internas da PYGUS.

5. DESTINATÁRIOS

Os dados dos Pacientes podem ser compartilhados com:

  • Parceiros: para o fornecimento de serviços especializados e autoridades públicas, conforme necessário
  • Órgãos da administração pública: para cumprir obrigações legais ou regulatórias
  • Nossos fornecedores e parceiros: no decurso de nossas atividades e para os mesmos fins descritos nesta política

6. DIREITOS DO TITULAR

Os Pacientes têm os seguintes direitos em relação aos Dados Pessoais:

  • Receber confirmação da existência de tratamento e acessá-los
  • Retificar ou modificar dados incompletos, inexatos ou desatualizados
  • Apresentar oposição ao tratamento de dados pessoais
  • Solicitar a anonimização, bloqueio ou eliminação de dados desnecessários ou excessivos
  • Solicitar a portabilidade dos dados a outro fornecedor
  • Receber informações sobre as entidades públicas e privadas com as quais a PYGUS compartilha dados pessoais
  • Receber informações sobre a possibilidade e consequências de não fornecer consentimento
  • Revogar o consentimento dado para qualquer finalidade

7. SEGURANÇA

A PYGUS adota medidas técnicas e organizacionais para garantir a segurança e confidencialidade dos dados pessoais dos Pacientes. A PYGUS também possui procedimentos para atuar em caso de suspeita de violação de segurança dos Dados Pessoais, fazendo a notificação pertinente às partes envolvidas e à autoridade aplicável.

8. ATUALIZAÇÕES

A PYGUS pode atualizar esta política de privacidade conforme necessário. As alterações serão comunicadas aos usuários através do aplicativo ou por e-mail. A utilização contínua dos nossos serviços após a publicação ou notificação das alterações implica a aceitação da Política de Privacidade atualizada. Se não concordar com alguma alteração, o Paciente tem o direito de fechar sua conta.
